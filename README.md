# Word-wrap

The class Wrapper, that has a single static function named wrap that takes two arguments, a string, and a column number. The function returns the string, but with line breaks inserted at just the right places to make sure that no line is longer than the column number. You try to break lines at word boundaries.

## Testing

- To test the code, run `rspec` from root of the folder using terminal.
- Rspec is used for the test, to install the gem file, run `bundle install`
